# AnimeFaceBoxes

Super fast anime face detection (> 200 fps for 512x512 resolution, RTX2080)

## Dependencies
- Python 3.6+ (Anaconda)
- PyTorch-1.0 +
- scipy, numpy, sklearn etc.
- OpenCV3 (Python)

## Usage
- build nms: sh make.sh
- labeled data: faceboxes (danbooru)
- train: MyTrain.py
- eval: eval.py
